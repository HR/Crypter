Welcome to Crypter

Crypter is a simple, convenient and secure encryption client.

This setup will walk you through setting up Crypter. Press the button below to get started.

Set a MasterPass for encryption

A MasterPassKey will be derived from the MasterPass. This will then be used to derive the keys used to encrypt your data.

NOTE: If lost, encrypted data will be unrecoverable! So, please store it safely.
